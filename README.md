# APEX - API Explorer for iRacing

> **The optimal line to your iRacing data**

## Overview

APEX is a [Bruno](https://www.usebruno.com) collection that uses the OAuth 2.1 Authorization Code Flow with PKCE (Proof Key for Code Exchange) to securely access the iRacing Data API. This method is designed for public/shared applications where multiple users can use the same collection without sharing credentials.
